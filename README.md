# sundials-rs

Safe, idiomatic Rust bindings to the
[SUNDIALS](https://computing.llnl.gov/projects/sundials) ODE/DAE solver
library (CVODE, CVODES, IDA, IDAS).

This workspace contains two crates:

| Crate | Description |
|---|---|
| [`sundials-rs`](sundials-rs/) | Safe, idiomatic wrapper — **this is the crate to use** |
| [`sundials-rs-sys`](sundials-rs-sys/) | Raw, unsafe FFI bindings generated by `bindgen` |

## Solvers

| Module | C library | Solves |
|---|---|---|
| `cvode` | CVODE | Explicit ODE IVP: `y' = f(t, y)` |
| `cvodes` | CVODES | CVODE + forward sensitivity `∂y/∂p` |
| `ida` | IDA | Implicit DAE IVP: `F(t, y, y') = 0` |
| `idas` | IDAS | IDA + forward sensitivity |

## Quick start

```toml
[dependencies]
sundials-rs = "0.1"

# Build SUNDIALS from source (requires cmake on PATH — no system library needed):
# sundials-rs = { version = "0.1", features = ["vendored"] }
```

```rust
use sundials_rs::cvode::{CvodeBuilder, Method};

let y0 = vec![1.0_f64];
let mut solver = CvodeBuilder::new(Method::BDF, &y0)
    .rtol(1e-8).atol(1e-10)
    .build(|_t, y, ydot| { ydot[0] = -y[0]; Ok(()) })
    .unwrap();

let (t, y) = solver.step(1.0).unwrap();
println!("y({t:.3}) = {:.8}  (exact {:.8})", y[0], (-t).exp());
```

## System requirements

**Without the `vendored` feature** — SUNDIALS >= 6.0 must be installed:

```bash
# Ubuntu / Debian
sudo apt install libsundials-dev

# macOS
brew install sundials
```

If SUNDIALS is installed in a non-standard location:

```bash
export SUNDIALS_DIR=/path/to/sundials/install
```

**With the `vendored` feature** — only `cmake` is required; SUNDIALS 7.4.0 is
downloaded and built automatically.

## Examples

Runnable examples live in [`sundials-rs/examples/`](sundials-rs/examples/):

```bash
cargo run --example robertson_cvode        # stiff ODE integration
cargo run --example robertson_cvodes_fsa   # ODE + forward sensitivities
cargo run --example robertson_ida          # same system in DAE/residual form
cargo run --example robertson_idas_fsa     # DAE + forward sensitivities
cargo run --example robertson_parallel     # parallel solves with rayon
```
